## 🌐 Real-Time Web Search Skill

You can add this skill to your app using either of the following two methods:

## 🚀 How to Setup & Add Skill

### Method 1: Import Local Folder
1. Download or clone this repository folder to your device.
2. Open Google AI Edge Gallery, go to **Manage Skills**, and click the **`+`** icon.
3. Select **Import local skill** and choose this repository folder.

### Method 2: Load via URL (Fastest & Recommended)
1. Open Google AI Edge Gallery, go to **Manage Skills**, and click the **`+`** icon.
2. Select **Load skill from URL**.
3. Paste the following live URL into the field and click **Add**:
   ```text
   https://saziudzaman-collab.github.io/web-search-skill/
   ```

---

## 🔑 How to Add API Key

This skill does not hardcode any private tokens. You need to add your own free API key to activate it:

1. **Get Key**: Get a free API key by signing up at [serper.dev](https://serper.dev).
2. **Open Secrets**: Under the **Custom Skills** or **Manage Skills** section, find `web-search` and click on the **`Secret`** button.
3. **Enter Key**: Paste your API key into the input field and click **Save**.
